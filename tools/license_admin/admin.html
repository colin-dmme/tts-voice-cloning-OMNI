Omni TTS License Admin

Tạo file license.json cho khách bằng mã máy trong tab Bản quyền của app.

Thông tin license

Chỉ dùng trên máy của bạn. Không gửi file private key cho khách.
Dán đúng mã máy khách gửi từ tab Bản quyền.
Sẵn sàng.

Cách dùng

  1. Khách mở app, vào tab Bản quyền.
  2. Khách bấm Sao chép mã máy và gửi mã đó cho bạn.
  3. Bạn dán mã máy vào form này, chọn ngày hết hạn và tính năng.
  4. Bấm Tạo và tải license.json.
  5. Gửi file license.json cho khách để họ bấm Nhập file license.
Trang này chỉ nên chạy trên máy admin. File license_private_key.pem là chìa khóa cấp bản quyền, mất file này là mất quyền kiểm soát license.
